Try Redis

发表于 2016-09-25   |  

redis

Redis 是一个开源（BSD许可）的，内存中的数据结构存储系统，它可以用作数据库、缓存和消息中间件。

save（存储）

set name redis

get（获取）

get name

del（删除）

del name

inrc（自增加一）

expire(存储时间)

expire name 120

ttl(查看剩余时间)

ttl name =>108

被删除后会输出-2，如果输出-1表示永不过期，如果重新set name value,ttl也会被reset

rpush(list末尾添加)

lpush(list头部添加)

lrange(获取list)

-1表示取到list的最后一位

llen(获取list长度)

llen friends => 3

lpop(移除list第一个)

lpop friends =>"Jesus"

rpop(移除list最后一个)

rpop friends

sadd(给set添加数据)

srem(从set移除数据)

srem heroes "Transfromers"

sismember(set中是否包含)

0表示不存在，1表示不存在

smembers(获取set所有值)

sunion(合并set)

Sorted Sets

sorted sets多了一个sorce,根据score排序

zragne

hset(存储对象)

hgetall(获取对象所有信息)

hmset(一次设置多个字段)

hget(获取单个字段)
